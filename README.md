What's included:

How to preview locally:
1. unzip and cd into the folder
2. npm install
3. npm run dev
4. open http://localhost:3000

Deploy to Vercel:
1. Create a GitHub repo and push this folder
2. Connect the repo to Vercel and click Deploy (Vercel auto-detects Next.js)
3. Optionally set environment variables for production later

To enable Podcast playback: replace PL_REPLACE_WITH_YOUR_PLAYLIST_ID in pages/index.js iframe src with your YouTube playlist ID.
